## maven
* maven是java的项目构建工具
* [入门]("https://juejin.cn/post/7051407711308627981")
* ant是java的另一个项目构建工具

## pom.xml
* groupId：公司或者是组织的id
* packaging：项目打包的类型，默认是 jar。一般 java 程序打包成 jar，web 程序打包成 war。
* version：项目的版本号，通常使用三位数字标识，例如 2.2.5。如果项目还在开发中，通常在版本后加上 SNAPSHOT。如果项目很稳定，已经发布，一般会在版本后加上 RELEASE。

## gradle
